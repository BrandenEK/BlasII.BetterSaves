# Blasphemous 2 Better Saves

<img src="https://img.shields.io/github/downloads/BrandenEK/BlasII.BetterSaves/total?color=872124&style=for-the-badge">

---

## Features
- Adds more save slots, for a maximum of 10
- Adds the ability to name the save files, with a command
- Displays the save file's last played date

## Available commands
| Command | Parameters | Description |
| ------- | ----------- | ------- |
| `slotname set` | NAME | Updates the name for the current save file |
| `slotname clear` | none | Resets the name for the current save file |
